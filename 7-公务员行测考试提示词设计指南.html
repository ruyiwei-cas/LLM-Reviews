行测题型框架

行政职业能力测验（行测）是公务员考试的核心内容。本页面介绍如何使用大模型生成各类行测题型的提示词框架。

言语理解与表达

片段阅读框架

  • 题材指定：明确文段主题领域和体裁
  • 能力指向：明确测试主旨概括、意图判断或细节理解
  • 难度控制：设置语言难度和逻辑复杂度
  • 干扰设计：明确干扰项设计思路
请以【资深公务员考试命题人】的身份，创建一道行测【片段阅读】题目，具体要求： 1. 文段主题：【宏观经济政策】，体裁为【政论文】风格 2. 文段长度：150-200字左右 3. 考查能力：【主旨概括】能力 4. 难度级别：【中等偏上】 5. 干扰项设计： - 一个【以偏概全】的选项 - 一个【无中生有】的选项 - 一个【混淆概念】的选项 - 一个【正确答案】 请提供完整题目、四个选项（A-D）及详细解析。解析需要说明正确选项的理由及其他选项错误的原因。

逻辑填空框架

  • 词语属性：指定词语类型（成语、熟语、多义词等）
  • 填空数量：单空、双空或多空
  • 考查重点：语境理解、搭配固定、感情色彩等
请设计一道行测【逻辑填空】题目，具体要求： 1. 题型：【双空】填空 2. 考查重点：【词语搭配】+【感情色彩】 3. 备选词语类型：以【成语】为主 4. 题目背景：围绕【环境保护】主题设计文段 5. 难度要求：设置明显的语境线索，但需通过细致辨析才能选出正确答案 请提供： - 完整题目（含填空） - 四个选项（A-D），每个选项包含两个词语 - 详细解析，需重点分析每组词语在语境中的适用性和搭配关系

判断推理

图形推理框架

  • 规律类型：位置、样式、数量、属性、组合等
  • 难度设定：单一规律或多重规律
  • 图形描述：清晰描述每个图形的构成要素
请设计一道行测【图形推理】题目，并提供详细的文字描述使读者能清晰理解图形内容。题目要求： 1. 规律类型：【旋转】+【元素变化】的组合规律 2. 难度级别：中等，设置两重规律，但有明显线索 3. 图形特征：使用【几何图形】（如圆形、三角形、正方形等）和【线条】元素 4. 问题结构：呈现5个图形作为题干，要求推测第6个图形 图形描述要求： - 使用精确的文字描述每个图形（如"一个三角形内部有两条垂直交叉的直线"） - 解释每组图形之间的变化规律 - 描述每个选项图形的具体特征 请提供完整题目描述、4个备选答案描述和详细解析。解析需说明图形变化的完整规律。

逻辑判断框架

  • 推理类型：必然性推理、可能性推理、加强削弱等
  • 题材背景：社会热点、政策法规、科学发现等
  • 论证结构：前提、结论和假设的设计
请设计一道行测【削弱型逻辑判断】题目，具体要求： 1. 题材背景：【城市交通拥堵治理】 2. 论证结构：论点+论据+隐含假设 3. 难度要求：中等，含有一定隐含假设 4. 削弱选项设计： - 一个【最有力削弱】选项（直指隐含假设） - 一个【部分削弱】选项（质疑论据可靠性） - 一个【无关选项】（看似相关但不影响论证） - 一个【加强选项】（错误方向干扰） 请提供完整论证过程、问题、四个选项和详细解析。解析需分析每个选项对论证的具体影响方式。

行测命题核心要素

设计行测提示词时，应当始终包含以下要素：

  • 明确题型分类：指定具体的行测题型和子类
  • 设定考查能力点：明确题目要测试的核心能力
  • 控制难度系数：通过具体参数设定题目难度
  • 干扰项设计思路：明确错误选项的设计方向

命题质量提升技巧

提高行测题目质量的关键策略：

  • 实际工作关联：将题目与公务员实际工作场景结合
  • 热点融入：适当融入时政热点和社会关注话题
  • 梯度设计：在同一类型中设置难度梯度，考查不同层次能力
  • 干扰项精细化：设计有针对性的干扰项，提高选择的区分度

案例：综合行测言语理解

作为国家公务员考试命题专家，请设计一套小型行测【言语理解】题组（共5题），覆盖不同题型，具体要求： 1. 题型分布： - 1道【主旨概括题】 - 1道【意图判断题】 - 1道【细节理解题】 - 1道【词句理解题】 - 1道【语句表达题】（如：语句排序或填入恰当的语句） 2. 主题设定： - 全部围绕【乡村振兴战略】这一主题 - 每道题使用不同角度的材料，但主题一致 - 材料应贴近政府工作实际，具有政策指导意义 3. 难度设置： - 整体难度中等偏上 - 5道题目难度递增 4. 其他要求： - 材料内容时效性强，反映近期政策导向 - 语言表述规范，符合公文特点 - 干扰项设计有区分度，能够测试考生的精准理解能力 请提供5道完整题目、参考答案和详细解析。解析应说明命题思路和选项设计意图。

此模板的优势：

  • 围绕单一主题设计系列题目，提高了命题的连贯性和专业深度
  • 覆盖言语理解各个题型，全面考查语言理解能力
  • 结合政策热点，贴近公务员工作实际
  • 设置难度梯度，提高了区分度

行测命题实战指南

使用大模型生成高质量行测题目的五大黄金法则：

  1. 目标精准化：明确指定题型、考点和能力测试目标
  2. 情境真实化：使用贴近公务员工作的真实场景
  3. 难度分级化：使用1-5级难度系数明确控制题目难度
  4. 干扰科学化：设计有针对性、有教育意义的干扰选项
  5. 评价标准化：提供客观、明确的评分标准和解题思路

记住：优质行测题目应该既能有效区分考生能力水平，又能反映公务员工作所需的实际技能和知识。
